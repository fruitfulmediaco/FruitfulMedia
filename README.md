## GitHub Pages Setup

This site is hosted using GitHub Pages. To view the live website, visit:
`https://fruitfulmediaco.github.io/FruitfulMedia/`

### To enable GitHub Pages:

1. Go to repository **Settings**
2. Scroll to **Pages** section
3. Under "Source", select `main` branch
4. Save the changes

The site will be published to your GitHub Pages URL.

## Customization

### Add Your Logo

1. Create an `assets` folder in the repository
2. Add your logo file as `logo.png` (or update the reference in `index.html`)
3. The logo will appear in the navigation bar

### Update Contact Information

Edit the **Contact** section in `index.html`:
- Replace email address
- Add your phone number
- Update social media links

### Add Portfolio Items

Expand the **Portfolio** section in `index.html` with your projects.

### Customize Colors

Edit `styles.css` to change the gradient colors:
- Primary gradient: `#667eea` and `#764ba2`
- Accent color: `#FFD700` (gold)
